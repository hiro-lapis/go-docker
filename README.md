## Intro
This is an Golang **ver1.19** dev-environment with hot reload(air).  
If you run `docker compose up`, go container has up with hot reload, instantly!

## Install Guide
###  up container with hot Reload

```shell
docker compose up
```
###  access container

```shell
docker exec -it go-docker_go_1 ash
```
